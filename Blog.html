Question-1: What are the different ways to select an element in the DOM?

  • getElementById - Selects an element by its ID.
  • getElementsByClassName - Selects elements by class name.
  • getElementsByTagName - Selects elements by tag name.
  • querySelector - Selects the first element matching a CSS selector.
  • querySelectorAll - Selects all elements matching a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML - Gets or sets HTML content, including tags.
  • innerText - Gets or sets visible text, ignoring hidden elements.
  • textContent - Gets or sets all text, including hidden elements, without formatting.

Question-3: What is event delegation in the DOM?

  • Event Delegation - A technique where a parent element handles events for its child elements using event.target.

Question-4: What is event bubbling in the DOM?

  • Event Bubbling - When an event starts from the target element and propagates up to its ancestors.
  • Example: Clicking a button inside a div triggers the button’s event first, then the div’s event.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create Element: let div = document.createElement("div");
  • Add Element: document.body.appendChild(div);
  • Remove Element: div.remove();
